## What Was Built

### BackupApp Handler (D-AM-05)

`BackupApp` / `BackupAppForTest` -- POST /devices/{serial}/apps/{pkg}/backup

- Exact `bu backup -f - -noshared -apk <pkg>` command (D-AM-05), pkg passed through `shellQuote`
- `validatePackage(w, r)` called BEFORE any ShellV2Stream call (REQ-AM-PKG-VALIDATE invariant)
- `WriteInFlight.CompareAndSwap(false, true)` single-flight gate; second caller gets 503 DEVICE_BUSY
- Bounded `context.Background()` + `cfg.APK.InstallTimeoutSeconds` timeout; backup survives client disconnect
- **W5 STRICT peek-before-headers**: reads exactly 4 bytes via `io.ReadFull` BEFORE committing any HTTP headers
  - `n != 4` (empty or partial stream): returns 500 `BACKUP_FAILED` JSON envelope -- headers NOT yet committed
  - `n == 4`: commits `Content-Type: application/octet-stream` + `Content-Disposition: attachment; filename="<pkg>.ab"` + writes peek + remainder
- Post-stream drain: best-effort `select` on `exitCh` / `backupCtx.Done()`
