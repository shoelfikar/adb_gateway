## What Was Built

### ListApps Handler (D-AM-01/03/04)

`ListApps` / `ListAppsForTest` -- GET /devices/{serial}/apps

- Exact `pm list packages` command with flags driven by `?include=` query parameter:
  - `include=""` (default): `pm list packages -3 -U -i --show-versioncode` (user-only, enabled)
  - `include="system"`: `pm list packages -U -i --show-versioncode` (system+user, enabled)
  - `include="disabled"`: `pm list packages -3 -d -U -i --show-versioncode` (user-only, disabled)
  - `include="all"`: `pm list packages -d -U -i --show-versioncode` (system+user, disabled-scope)
- W4 compliance: `includeSystem`/`includeDisabled` derived from query param and passed to `ParsePMList`; handler NEVER mutates parser output post-hoc
- `?name=<substr>` case-insensitive substring filter on Package field (D-AM-03)
- Shell failure maps to 500 LIST_FAILED with slog.Warn

### GetAppDetails Handler (D-AM-02/04)

`GetAppDetails` / `GetAppDetailsForTest` / `AppDetailsForTest` -- GET /devices/{serial}/apps/{pkg}

- `validatePackage(w, r)` called BEFORE any shell call (REQ-AM-PKG-VALIDATE invariant)
- `dumpsys package 'pkg'` (shellQuote single-quoted)
- Detects "Unable to find package" in dumpsys output for 404 PACKAGE_NOT_FOUND
- `ParseDumpsysPackage(out, pkg)` lenient parser
- `?include_size=1` opt-in: `du -sk /data/data/'pkg'` adds `total_size_bytes` to response (~500ms-2s overhead for large data dirs)
- Shell failure maps to 500 LIST_FAILED

### UninstallApp Handler (D-AM-08)

`UninstallApp` / `UninstallAppForTest` -- DELETE /devices/{serial}/apps/{pkg}

- `validatePackage(w, r)` called BEFORE any shell call
- `WriteInFlight.CompareAndSwap(false, true)` single-flight gate (Pitfall 9); second caller gets 503 DEVICE_BUSY
- Bounded `context.Background()` + `cfg.APK.InstallTimeoutSeconds` timeout (mirrors APK install D-08); uninstall survives client disconnect
- `ShellV2Stream` for Pitfall 4: checks BOTH exit code AND stdout for "Failure" substring
- Error mapping per D-AM-08:
  - "not installed for" substring (case-insensitive combined stdout+stderr) -> 404 PACKAGE_NOT_FOUND
  - Other failures -> 500 UNINSTALL_FAILED with truncated stderr/stdout (256 bytes max)
